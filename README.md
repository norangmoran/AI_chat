# AI_chat

이 레포지토리는 **「챗GPT API를 활용한 챗봇 만들기 개정판」** 도서를 기반으로  
AI 챗봇 개발을 실습 후 응용하기 위해 다양한 AI 툴 및 기술 스택을 연습하려고 만들어졌습니다.

---

## 📘 학습 목적

- OpenAI(ChatGPT) API 활용법 이해
- Python 기반 챗봇 개발 실습
- Flask를 이용한 간단한 웹 서버 구축
- MongoDB / VectorDB 등을 활용한 데이터 저장 및 검색
- RAG(Retrieval-Augmented Generation) 구조 연습
- AI 서비스 개발 전반의 흐름 익히기

---

## 🛠 기술 스택

- **Python 3.12.7**
- **Flask 3.1.0**
- **OpenAI API**
- **MongoDB (Docker)**
- **Vector DB (Docker)**
- **Node.js / React (vite)**

## 🚀 목표 기능

- React를 이용한 웹 UI 구현
